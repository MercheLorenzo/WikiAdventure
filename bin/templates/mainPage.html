Welcome to the beta 1.0 of the game

How to play

The goal is simple, you need to go as fast as possible from the start page to the end page, but you can only navigate with the link on the wikipedia page. During the voting phase you can suggest your wikipedia page title in the swipe menu. At the end of the voting phase the server will randomly pick a start page and an end page and the game will begin.
